# Image-Viewer
A Java written image viewer that has zoom-in, zoom-out and rotation functions.
Use this viewer only for imageA.jpg because the parameters are not adjusted automatically.

## Necessary Package(s)
OpenCV

## Suggestions
Change the file path if your structure is differnet from mine.
Please set up your .idea configurations by yourself.
